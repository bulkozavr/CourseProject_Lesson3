# Locale Reference

## Conventions

- **Server:** Node `http` module, no Express
- **Port:** 3000
- **API prefix:** `/api/`
- **Static dir:** `./public/`
- **Data file:** `locales.json` — read at startup, no DB
- **Code style:** ES6+ (Node 18+), `const`/`let`, arrow functions
- **Tests:** `.sh` in `test/` using `curl`. `.js` using `node` assert
- **Commit style:** Conventional Commits (`feat:`, `fix:`, `chore:`, `test:`)
